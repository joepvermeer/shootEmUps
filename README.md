# shootEmUps


Welkom bij onze implementatie van Shoot 'em up! 

het programma gebruikt de dependencies uit package.yaml, 
dus om het programma te builden en te runnen kan je het 
commando; "stack build && stack run" gebruiken

controls: 
de controls van het programma zijn redelijk straight forward en worden in het programma ook toegelicht 
enter   : na het invullen van je naam 
s       : starten van de game voor het eerst 
p       : pauzeren/continue voor de game 
r       : restarten van het spel 
esc     : sluiten van het spel 
spatie  : schieten 
pijltjes: voor het bewegen van de speler 


de scores worden automatisch naar scores.txt geschreven 
als je lokaal een 'vers' score bestand wil hebben, kan je hem verwijderen 
en dan wordt bij de eerste keer runnen van de game een nieuw bestand aangemaakt onder "scores.txt"
